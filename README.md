# dotfiles

Managed by [chezmoi](https://www.chezmoi.io/quick-start/).

Run the command to install the dotfiles:

```
curl -sfL https://raw.githubusercontent.com/jamesdoc/dotfiles/main/.setup.sh | bash
```

## In the box

- Homebrew
- Chez Moi
- Starship

Plus a load of sensible macos defaults


## 

- `chezmoi update` to pull and apply changes
- `chezmoi apply` to reapply config
- `chezmoi init` to regenerate chezmoi config
